### CDN (Content Delivery Network)

**Distribusi konten** ke edge servers di seluruh dunia untuk latency rendah.

**Fitur Utama:**
- ✅ 200+ PoP (Point of Presence) global
- ✅ Caching static & dynamic content
- ✅ SSL/TLS support
- ✅ DDoS protection
- ✅ Real-time analytics
- ✅ Image optimization

**Pricing:**
- **Traffic-based** - Rp 500/GB
- **Request-based** - Rp 5.000 per 10.000 requests

**Use Cases:**
- Media streaming
- E-commerce product images
- Software downloads
- Website assets (CSS, JS, images)

---

### Private Network

**Jaringan privat** untuk koneksi antar VPS/server yang aman.

**Fitur Utama:**
- ✅ Isolated network (tidak terekspos internet)
- ✅ Low latency (< 1ms dalam region yang sama)
- ✅ High bandwidth (hingga 10 Gbps)
- ✅ Free traffic antar VPS dalam network

**Pricing:**
- **Free** untuk Private Network dalam region yang sama

**Use Cases:**

- Backup replication
- Multi-tier applications

**Contoh Usage:**
```bash
# Login
dalang login

# List VPS
dalang vps list

# Create VPS
dalang vps create --plan standard --os ubuntu-24.04 --region jakarta

# Scale VPS
dalang vps scale <vps-id> --plan business

dalang cost estimate --monthly
```
