## Apa itu VPS?

**VPS (Virtual Private Server)** adalah server virtual yang berjalan di atas infrastruktur cloud. VPS memberikan kamu:

- ✅ **Root Access Penuh** - Kontrol lengkap seperti memiliki server fisik
- ✅ **Resource Dedicated** - CPU, RAM, dan storage yang tidak dibagi dengan user lain
- ✅ **Isolasi Penuh** - Lingkungan terisolasi untuk keamanan maksimal
- ✅ **Scalability** - Upgrade resource kapan saja sesuai kebutuhan
- ✅ **Cost Effective** - Biaya lebih murah dibanding dedicated server

### Analogi Sederhana

Bayangkan sebuah gedung apartemen:

| Tipe Hosting | Analogi | Karakteristik |
|-------------|---------|---------------|
| **Shared Hosting** | Kamar kos bersama | Share fasilitas dengan banyak orang, terbatas |
| **VPS** | Apartemen pribadi | Punya ruang sendiri, bebas modifikasi, resource dedicated |
| **Dedicated Server** | Rumah tapak pribadi | Satu gedung untuk sendiri, kontrol penuh |

## Security Best Practices

Setelah deploy VPS, lakukan hal berikut untuk keamanan:

### 1. Update Sistem

```bash
# Ubuntu/Debian
sudo apt update && sudo apt upgrade -y

# CentOS/Rocky
sudo dnf update -y
```

### 2. Setup Firewall

```bash
# UFW (Ubuntu/Debian)
sudo ufw allow 22/tcp    # SSH
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
sudo ufw enable
```

### 3. Disable Root Login

```bash
# Edit SSH config
sudo nano /etc/ssh/sshd_config

# Ubah:
PermitRootLogin no
PasswordAuthentication no
```

### 4. Setup SSH Key

```bash
# Generate key pair
ssh-keygen -t ed25519

# Copy ke VPS
ssh-copy-id user@vps-ip
```

### 5. Install Fail2Ban

```bash
# Install
sudo apt install fail2ban -y

# Enable
sudo systemctl enable fail2ban
sudo systemctl start fail2ban
```

### 6. Enable Automatic Security Updates

```bash
# Ubuntu/Debian
sudo apt install unattended-upgrades -y
sudo dpkg-reconfigure --priority=low unattended-upgrades
```
